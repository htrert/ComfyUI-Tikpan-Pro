# AI 短剧 / 漫剧 / TikTok 内容工厂工作流

这个画布不要做成通用版 ComfyUI。个人开发者更适合做“半无限画布”：用户输入一个目标，系统自动铺开一套垂直内容生产流程。

## 产品定位

Canvas Model Studio 的核心不是让用户自由搭任意节点，而是帮助用户完成一个结果：

```text
小说 / 短剧梗概 / 产品信息
-> 项目 Brief
-> 角色与世界观资产
-> 九宫格分镜
-> 图片 / 视频 / 音乐 / 配音生成
-> 字幕 / 封面 / 标题
-> 导出素材包或发布任务
```

ComfyUI 负责本地 GPU 执行层，适合 SDXL、Flux、LoRA、局部重绘、批量图生图。云端 API 负责 GPT、视频、音乐、长上下文 Agent 和多模态理解。画布负责项目、资产、调度、计费和长期上下文。

## 1. AI 短剧 / 漫剧工厂

适用场景：小说改编、AI 漫剧、AI 动漫短剧、角色账号、连续剧集。

推荐流程：

```text
原文/梗概
-> 项目 Brief
-> 世界观 Bible
-> 角色卡
-> 场景卡
-> 九宫格分镜
-> 关键帧图
-> 镜头视频化
-> BGM / 配音 / 字幕
-> 导出发布清单
```

关键数据：

- `Project`: 项目名、题材、平台、画幅、目标用户、更新频率。
- `Character`: 名字、身份、外观 DNA、服装、参考图、禁用变化。
- `WorldBible`: 世界观、时代、场景、道具、色调、镜头语言。
- `StoryboardShot`: 镜头编号、画面描述、人物、场景、台词、时长、生成状态。
- `Asset`: 图片、视频、音频、字幕、封面、来源任务。

九宫格建议：

```text
1 钩子
2 主角状态
3 冲突出现
4 误会/证据
5 情绪特写
6 对手压迫
7 反转
8 高潮
9 下集钩子
```

## 2. 小说推文 / TikTok 切片

适用场景：小说号、推文号、英文 TikTok、批量内容切片。

推荐流程：

```text
小说片段
-> 高潮点提取
-> 3 秒钩子
-> 60 秒旁白脚本
-> 情绪关键帧
-> 封面冲突图
-> 字幕和标题
-> 多语言版本
```

重点不是生成精美单图，而是持续生产高完播素材。每个片段都要记录：

- 钩子类型：背叛、复仇、打脸、反转、身份曝光。
- 旁白结构：开场、背景、冲突、反转、下集钩子。
- 转化指标：点击率、完播率、收藏率、评论关键词。

## 3. TikTok 电商内容工厂

适用场景：跨境电商、带货短视频、产品广告、素材矩阵。

推荐流程：

```text
产品图 / SKU 信息
-> 产品卖点
-> 3 组脚本角度
-> 产品场景图
-> AI 模特 / 使用场景
-> 3-8 秒视频镜头
-> 字幕 / 标题 / 封面
-> A/B 测试矩阵
```

每个 SKU 至少测试：

- 3 个卖点角度。
- 2 个封面。
- 3 个标题。
- 2 个视频节奏。

## 4. 后端应该怎么演进

当前 MVP 已经有：

- `/api/content-factory/workflows`: 工作流模板。
- `/api/content-factory/plan`: 根据模板生成画布节点。
- `/api/projects`: 用户项目保存。
- `/api/assets`: 用户资产保存。
- `/api/admin/config`: 渠道、模型、参数、计费配置。

后续建议新增这些表：

```text
projects
assets
characters
world_bibles
storyboard_shots
workflow_runs
workflow_tasks
model_routes
usage_records
billing_events
```

任务状态建议：

```text
draft -> queued -> running -> succeeded
                      |
                      -> failed -> retrying -> failed_final
```

每个任务必须保存：

- 输入参数。
- 使用模型。
- 渠道。
- 成本估算。
- 输出资产。
- 错误信息。
- 重试次数。

## 5. 个人开发者优先级

先做小闭环：

```text
输入小说
-> 自动九宫格分镜
-> 生成角色/场景/关键帧
-> 保存资产
-> 导出素材包
```

暂时不要优先做：

- 多人实时协作。
- 通用节点编辑器。
- 完整 Agent OS。
- 自建视频模型。
- 大规模本地推理适配。

先卖结果，再扩工具。
