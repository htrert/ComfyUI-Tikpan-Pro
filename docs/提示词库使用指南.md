# 提示词库使用指南

> 从 9 个精选 GitHub 仓库同步优质提示词，一键注入到生成节点

---

## 快速上手

### 第一步：同步提示词库

在 ComfyUI 中添加 **"提示词库管理器"** 节点：

```
操作: 同步全部仓库
```

点击执行，等待 10-30 秒（需要联网），控制台会显示同步进度。

**同步完成后会显示：**
```
✅ 同步完成：287 张卡片
```

### 第二步：选择提示词

添加 **"提示词选择器"** 节点：

```
选择提示词: 1. 商品主图拍摄
过滤_标签: gpt-image-2
```

将 **"提示词文本"** 输出连接到你的图像/视频生成节点的 prompt 输入。

**完整工作流示例：**
```
[提示词选择器] → 提示词文本 → [GPT Image 2 生图] → [保存图像]
```

## 工作原理

### 数据来源

提示词库从 9 个精选 GitHub 仓库自动抓取：

| 仓库 | 用途 |
|------|------|
| awesome-nano-banana-pro-prompts | Nano Banana 图像提示词 |
| awesome-gpt-image-2-API-and-Prompts | GPT-Image-2 图像提示词 |
| awesome-seedance-2.0-prompts | Seedance 视频提示词 |
| ai-mixed-cut | AI 混剪工作流 |
| ai-highlight-clip | 高光切片工作流 |
| ai-ttv-workflow | 文生视频流程 |
| video-mover | 视频搬运/适配 |
| BigBanana-AI-Director | AI 导演/多智能体 |

### 解析逻辑

1. 从 GitHub 获取仓库的 README.md
2. 按 `## 标题` 切分成多个段落
3. 每个段落提取：
   - **标题**：作为提示词名称
   - **提示词内容**：优先提取代码块 → blockquote → 长段落
   - **标签**：自动识别 `Tags: xxx` 或从仓库继承
4. 保存到本地 `data/prompts-library/` 目录

### 本地存储

同步后的数据保存在：
```
data/prompts-library/
├── repo_YouMind-OpenLab__awesome-nano-banana-pro-prompts.json
├── repo_EvoLinkAI__awesome-gpt-image-2-API-and-Prompts.json
└── ...
```

**离线使用**：同步后可断网使用，数据永久保存在本地。

## 三个节点详解

### 1. 提示词库管理器

**用途**：同步和管理提示词库

**输入参数：**
- `操作`：查看状态 / 同步全部仓库
- `同步单个仓库`：选择特定仓库单独同步

**输出：**
- `状态报告`：同步结果和统计信息

**使用场景：**
- 首次使用时同步
- 每周更新一次（仓库会持续更新）

---

### 2. 提示词选择器

**用途**：从下拉列表选择提示词

**输入参数：**
- `选择提示词`：从所有提示词中选择（显示前 200 个）
- `过滤_仓库`：只显示特定仓库的提示词
- `过滤_标签`：按标签筛选（image / video / nano-banana 等）
- `搜索关键词`：输入关键词搜索
- `显示详情`：是否在控制台显示完整信息

**输出：**
- `提示词文本`：提示词内容（连接到生成节点）
- `卡片标题`：提示词标题
- `卡片详情`：包含来源、标签、链接等信息

**适合场景**：浏览和选择提示词

---

### 3. 提示词搜索

**用途**：高级搜索功能

**输入参数：**
- `搜索关键词`：输入关键词（必填）
- `过滤_仓库`：按仓库过滤
- `过滤_标签`：按标签过滤
- `最多返回数量`：限制结果数量（1-50）
- `结果索引`：选择第几个结果（0 = 第一个）

**输出：**
- `提示词文本`：选中的提示词内容
- `卡片标题`：选中的提示词标题
- `搜索结果列表`：所有匹配结果的列表
- `结果总数`：匹配的总数量

**适合场景**：快速查找特定类型的提示词

## 实用技巧

### 按模型类型过滤

不同模型适合不同提示词：

```
GPT-Image-2  → 过滤_标签: gpt-image-2
Nano Banana  → 过滤_标签: nano-banana
视频生成      → 过滤_标签: video 或 seedance
```

### 组合过滤

同时使用多个条件缩小范围：

```
过滤_仓库: EvoLinkAI/awesome-gpt-image-2-API-and-Prompts
过滤_标签: image
搜索关键词: 商品
```

### 自定义提示词库

在 `data/prompts-library/` 创建自己的 JSON 文件：

```json
{
  "slug": "my-prompts",
  "updatedAt": "2026-05-31T10:00:00Z",
  "cards": [
    {
      "id": "my-1",
      "repo": "my-prompts",
      "title": "我的提示词",
      "prompt": "这里是提示词内容...",
      "tags": ["custom"],
      "body": "",
      "url": ""
    }
  ]
}
```

## 常见问题

**Q: 同步失败怎么办？**

检查网络连接，稍后重试。可以尝试同步单个仓库定位问题。

**Q: 提示词列表为空？**

需要先使用"提示词库管理器"同步。

**Q: 如何查看提示词完整内容？**

勾选"显示详情"，执行后查看控制台输出。

**Q: 提示词太多，如何快速找到想要的？**

使用"提示词搜索"节点，输入具体关键词（如 "商品"、"人物"）。

## 工作流示例

### 图像生成

```
[提示词搜索]
  搜索关键词: 商品摄影
  过滤_标签: gpt-image-2
  ↓
[GPT Image 2 官方生图]
  生成指令: (连接提示词文本)
  ↓
[保存图像]
```

### 视频生成

```
[提示词选择器]
  选择提示词: 15. 产品展示视频
  过滤_标签: video
  ↓
[Veo 3.1 视频生成]
  提示词: (连接提示词文本)
  ↓
[保存视频]
```
