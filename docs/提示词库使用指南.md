# 提示词库功能使用指南

## 📚 功能概述

ComfyUI-Tikpan-Pro 提示词库功能可以从 9 个精选的 GitHub 仓库自动同步优质提示词，帮助你快速找到适合的提示词模板，提升创作效率。

### 核心特性

- ✅ **9个精选仓库**：涵盖图像、视频、工作流等多个领域
- ✅ **智能解析**：自动从 README 提取提示词和标签
- ✅ **本地存储**：同步后可离线使用
- ✅ **强大过滤**：按仓库、标签、关键词快速筛选
- ✅ **即插即用**：输出文本可直接连接到生成节点

## 🚀 快速开始

### 第一步：同步提示词库

1. 在 ComfyUI 画布中添加 **"工具｜提示词库管理器"** 节点
2. 将 **"操作"** 参数设置为 **"同步全部仓库"**
3. 执行节点（需要联网，约 10-30 秒）
4. 查看控制台输出，确认同步成功

```
示例输出：
============================================================
📚 提示词库同步完成
============================================================
成功: 9/9 个仓库
总卡片数: 287

详细结果:
------------------------------------------------------------
✅ YouMind-OpenLab/awesome-nano-banana-pro-prompts: 45 张卡片
✅ EvoLinkAI/awesome-gpt-image-2-API-and-Prompts: 38 张卡片
...
============================================================
```

### 第二步：选择提示词

#### 方式 A：使用提示词选择器（推荐新手）

1. 添加 **"工具｜提示词选择器"** 节点
2. 从 **"选择提示词"** 下拉列表中选择想要的提示词
3. 可选：设置过滤条件
   - **过滤_仓库**：只显示特定仓库的提示词
   - **过滤_标签**：按标签筛选（如 image、video、nano-banana）
   - **搜索关键词**：输入关键词搜索
4. 将 **"提示词文本"** 输出连接到图像/视频生成节点的 prompt 输入

#### 方式 B：使用提示词搜索（推荐高级用户）

1. 添加 **"工具｜提示词搜索"** 节点
2. 在 **"搜索关键词"** 中输入关键词（如 "商品"、"人物"、"风景"）
3. 设置过滤条件（可选）
4. 设置 **"结果索引"** 选择第几个结果（0 = 第一个）
5. 将 **"提示词文本"** 输出连接到生成节点

## 📖 节点详细说明

### 1. 提示词库管理器

**节点名称**：`工具｜提示词库管理器`

**功能**：管理提示词库的同步和状态查看

**输入参数**：
- **操作**：选择要执行的操作
  - `查看状态`：查看当前提示词库状态和统计信息
  - `同步全部仓库`：从 GitHub 同步所有仓库
- **同步单个仓库**（可选）：选择要同步的单个仓库

**输出**：
- **状态报告**：操作结果的详细报告（STRING）

**使用场景**：
- 首次使用时同步提示词库
- 定期更新提示词库（建议每周一次）
- 查看库的统计信息

**示例工作流**：
```
[提示词库管理器]
  操作: 同步全部仓库
  ↓
  状态报告 → [显示文本节点]
```

---

### 2. 提示词选择器

**节点名称**：`工具｜提示词选择器`

**功能**：从下拉列表中选择提示词

**输入参数**：
- **选择提示词**：从所有提示词中选择（显示前 200 个）
- **过滤_仓库**（可选）：按仓库过滤
- **过滤_标签**（可选）：按标签过滤
- **搜索关键词**（可选）：输入关键词搜索
- **显示详情**（可选）：是否在控制台显示详细信息

**输出**：
- **提示词文本**：提示词内容（STRING）
- **卡片标题**：提示词标题（STRING）
- **卡片详情**：包含来源、标签、链接等完整信息（STRING）

**使用场景**：
- 浏览和选择提示词
- 快速预览提示词内容
- 了解提示词的来源和标签

**示例工作流**：
```
[提示词选择器]
  选择提示词: 1. 商品主图拍摄
  过滤_标签: gpt-image-2
  ↓
  提示词文本 → [GPT Image 2 生图节点]
```

---

### 3. 提示词搜索

**节点名称**：`工具｜提示词搜索`

**功能**：高级搜索和过滤提示词

**输入参数**：
- **搜索关键词**：输入搜索关键词（必填）
- **过滤_仓库**（可选）：按仓库过滤
- **过滤_标签**（可选）：按标签过滤
- **最多返回数量**（可选）：限制返回结果数量（1-50，默认 10）
- **结果索引**（可选）：选择第几个结果（0 = 第一个）

**输出**：
- **提示词文本**：选中的提示词内容（STRING）
- **卡片标题**：选中的提示词标题（STRING）
- **搜索结果列表**：所有匹配结果的列表（STRING）
- **结果总数**：匹配的总数量（INT）

**使用场景**：
- 按关键词快速查找提示词
- 批量查看搜索结果
- 通过索引切换不同结果

**示例工作流**：
```
[提示词搜索]
  搜索关键词: 人物肖像
  过滤_标签: image
  最多返回数量: 10
  结果索引: 0
  ↓
  提示词文本 → [图像生成节点]
  搜索结果列表 → [显示文本节点]（查看所有结果）
```

## 💡 使用技巧

### 1. 按模型类型过滤

不同的生成模型适合不同的提示词风格：

- **GPT-Image-2**：使用 `gpt-image-2` 标签过滤
- **Nano Banana**：使用 `nano-banana` 标签过滤
- **视频生成**：使用 `video` 或 `seedance` 标签过滤

### 2. 组合过滤条件

可以同时使用多个过滤条件缩小范围：

```
过滤_仓库: EvoLinkAI/awesome-gpt-image-2-API-and-Prompts
过滤_标签: image
搜索关键词: 商品
```

### 3. 定期更新

提示词仓库会不断更新，建议：
- 每周运行一次"同步全部仓库"
- 关注新增的提示词和标签

### 4. 离线使用

同步后的提示词保存在本地，可以：
- 断网后继续使用
- 手动编辑 `data/prompts-library/*.json` 文件
- 添加自己的私有提示词库

### 5. 自定义提示词库

在 `data/prompts-library/` 目录下创建自己的 JSON 文件：

```json
{
  "slug": "private/my-prompts",
  "updatedAt": "2026-05-31T10:00:00Z",
  "cards": [
    {
      "id": "my-1",
      "repo": "private/my-prompts",
      "title": "我的自定义提示词",
      "prompt": "这里是提示词内容...",
      "tags": ["custom", "image"],
      "body": "额外的说明信息",
      "url": ""
    }
  ]
}
```

## 🔧 常见问题

### Q1: 同步失败怎么办？

**可能原因**：
- 网络连接问题
- GitHub 访问受限
- 仓库被删除或重命名

**解决方法**：
1. 检查网络连接
2. 尝试同步单个仓库定位问题
3. 查看控制台错误信息
4. 稍后重试

### Q2: 提示词列表为空？

**原因**：尚未同步提示词库

**解决方法**：
1. 添加"提示词库管理器"节点
2. 选择"同步全部仓库"
3. 执行节点等待同步完成

### Q3: 如何查看提示词的完整内容？

**方法 1**：使用提示词选择器
- 勾选"显示详情"
- 执行节点后查看控制台输出

**方法 2**：查看输出
- 将"卡片详情"输出连接到文本显示节点

**方法 3**：查看源文件
- 打开 `data/prompts-library/*.json` 文件

### Q4: 提示词数量太多，如何快速找到想要的？

**建议**：
1. 使用"提示词搜索"节点
2. 输入具体的关键词（如 "商品"、"人物"、"风景"）
3. 设置标签过滤（如 image、video）
4. 查看"搜索结果列表"输出

### Q5: 可以添加自己的提示词仓库吗？

**当前版本**：仓库列表是硬编码的，需要修改代码

**未来计划**：将支持通过配置文件添加自定义仓库

**临时方案**：手动创建 JSON 文件（参考"自定义提示词库"章节）

## 📊 数据存储

### 存储位置

```
ComfyUI-Tikpan-Pro/
└── data/
    └── prompts-library/
        ├── repo_YouMind-OpenLab__awesome-nano-banana-pro-prompts.json
        ├── repo_EvoLinkAI__awesome-gpt-image-2-API-and-Prompts.json
        └── ...
```

### 文件格式

每个 JSON 文件包含：
- `slug`：仓库标识
- `sourceUrl`：README 来源 URL
- `updatedAt`：最后更新时间
- `cards`：提示词卡片数组
  - `id`：唯一标识
  - `repo`：所属仓库
  - `title`：标题
  - `prompt`：提示词内容
  - `body`：上下文信息
  - `tags`：标签数组
  - `url`：GitHub 链接
