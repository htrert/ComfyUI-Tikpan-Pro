> 更新日期：2026-05-11  
> 用途：快速查找每个节点的能力、关键输入和输出，方便 ComfyUI 使用，也方便后续 Web 网站做表单和计费映射。

## 网站化字段映射建议

| 后台表 | 建议字段 |
| --- | --- |
| 模型表 `models` | `model_key`、`display_name`、`category`、`provider`、`supports_image`、`supports_video`、`supports_audio`、`enabled` |
| 参数表 `model_parameters` | `model_key`、`field_key`、`label`、`type`、`required`、`default_value`、`options_json`、`sort_order` |
| 供应商渠道表 `provider_channels` | `provider`、`base_url`、`api_key_encrypted`、`priority`、`weight`、`enabled`、`timeout_seconds` |
| 价格表 `pricing_rules` | `model_key`、`unit_type`、`base_cost`、`user_price`、`currency`、`resolution_multiplier`、`duration_multiplier` |
| 任务表 `generation_tasks` | `task_id`、`user_id`、`model_key`、`status`、`request_json`、`provider_task_id`、`cost_amount`、`result_url`、`error_message` |
| 用户余额流水表 `wallet_ledger` | `user_id`、`task_id`、`amount`、`type`、`balance_after`、`remark` |

核心原则：ComfyUI 里的节点参数可以直接变成网站表单字段，但供应商密钥、渠道选择、真实成本和扣费逻辑必须放在后台。

## Vidu Q3 新增节点

| 节点 | 主要用途 | 关键输入 | 输出 |
| --- | --- | --- | --- |
| `TikpanViduQ3Node` | `viduq3` 参考生视频，支持 540p/720p/1080p | `api_key`、`prompt`、`resolution`、`duration`、`参考图1-7` 或 `图片URL列表`、`audio`、`seed` | 本地路径、任务 ID、云端直链、日志、`VIDEO` |
| `TikpanViduQ3MixNode` | `viduq3-mix` 参考生视频，多机位一致性更好，支持 720p/1080p | `api_key`、`prompt`、`resolution`、`duration`、`参考图1-7` 或 `图片URL列表`、`audio`、`seed` | 本地路径、任务 ID、云端直链、日志、`VIDEO` |
| `TikpanViduQ3TurboNode` | `viduq3-turbo` 多模式视频，支持文生/图生/首尾帧/参考生视频 | `生成模式`、`api_key`、`prompt`、`resolution`、`duration`、`首帧图`、`尾帧图`、`参考图1-7`、`图片URL列表` | 本地路径、任务 ID、云端直链、日志、`VIDEO` |

稳定性设计：这三个节点提交前会写入 `recovery/vidu_q3_video`；POST 创建任务不做 5xx 盲重试，避免上游已扣费后重复提交；同参数命中本地缓存时不会再次请求上游。
