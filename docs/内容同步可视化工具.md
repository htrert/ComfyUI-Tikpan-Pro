# 内容同步可视化工具

本项目提供一个本地可视化工具，用于拖入或粘贴文章内容，并导出/同步到多个内容平台。

## 启动

```bash
python scripts/content_sync_tool.py
```

如果当前系统的 `python` 不可用，可以使用项目里的虚拟环境：

```bash
./web_app/.venv/Scripts/python.exe scripts/content_sync_tool.py
```

启动后会自动打开浏览器，默认地址：

```text
http://127.0.0.1:8765/
```

## 当前支持

- 拖入或选择一个或多个 `.md` / `.txt` / `.html` 文章文件
- 在文件列表中切换、编辑或移除文章
- 手动编辑标题和正文
- 保存当前文章草稿到 `.feishu/content_sync_tool/drafts/`
- 导出当前文章或批量导出全部文章到 `.feishu/content_sync_tool/exports/`
  - 飞书文档
  - 微信公众号
  - 知识星球
  - 抖音长文章
  - Bilibili 长文章
- 同步当前文章到飞书文档
- 生成微信公众号 HTML 排版文件，用于浏览器半自动复制发布
- 使用微信公众号开放平台 API 创建草稿

## 飞书同步配置

飞书同步继续复用项目根目录的 `.feishu_config.json`：

```bash
cp .feishu_config.example.json .feishu_config.json
```

填好 `app_id` 和 `app_secret` 后，可以直接在工具界面选择 `.feishu_config.json` 里已有的 `files` 映射，也可以手动填写目标飞书文档 Token/ID，再点击“同步当前到飞书”。

注意：同步到飞书会清空目标文档旧内容，再写入当前文章。

## 微信公众号发布

### 半自动浏览器流程

在工具界面点击“生成公众号 HTML”，会把当前文章转换为适合公众号编辑器复制的 HTML 文件，保存到：

```text
.feishu/content_sync_tool/exports/wechat_html/
```

然后点击“打开公众号后台”，在浏览器登录公众号后台，新建图文后从生成的 HTML 页面复制排版内容到编辑器，再手动检查封面、摘要、图片和发布设置。

### 开放平台草稿 API

如果公众号已开启开发配置，可以复制示例配置：

```bash
cp .wechat_config.example.json .wechat_config.json
```

填入 `app_id`、`app_secret`，并在公众号素材库上传封面图获得 `thumb_media_id`。工具界面填写 `thumb_media_id` 后点击“创建公众号草稿”，会调用公众号草稿 API 创建图文草稿。

注意：公众号草稿 API 只能创建草稿，不会直接群发；发布前仍建议进入公众号后台人工预览和确认。

## 其他平台说明

知识星球、抖音长文章、Bilibili 长文章当前先做“适配文稿导出”。这些平台的稳定公开发布 API 能力不同，直接全自动发布容易受接口权限或平台风控影响。后续如果你提供对应开放平台凭证或确定要走浏览器半自动发布，可以继续扩展为一键发布/半自动填充。
