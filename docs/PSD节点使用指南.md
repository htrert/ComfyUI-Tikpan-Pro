# PSD 节点使用指南

## 📚 三个节点的区别

### 1. PSD 文件保存器
**功能**：简单保存为 PSD 格式（不分层）

**使用场景**：
- 只需要 PSD 格式
- 不需要分层编辑
- 快速转换

**工作流**：
```
[生图节点] → [PSD 文件保存器] → output/xxx.psd
```

---

### 2. 智能分层 PSD 生成器 ⭐推荐
**功能**：AI 自动识别元素并分层

**使用场景**：
- 需要在 Photoshop 中编辑
- 分离产品、背景、文字
- 商业级分层需求

**三档选择**：
- **经济档**（300MB，5-10秒）：rembg + OpenCV，适合简单商品图
- **标准档**（2.4GB，15-30秒）：SAM2 + EasyOCR，适合复杂场景 ⭐推荐
- **极致档**（5GB+，60-120秒）：SAM2 + LaMa，被遮挡区域智能补全

**工作流**：
```
[生图节点] → [智能分层 PSD 生成器]
                分层档位: 标准档
                检测文字: 是
                ↓
              output/xxx.psd（多图层）
              预览图
              分层日志
```

---

### 3. PSD 模型预下载器（可选）
**功能**：提前下载 AI 模型

**使用场景**：
- 首次使用智能分层前
- 避免等待下载
- 批量下载所有档位

**说明**：
- **不是必须的**：智能分层节点会自动下载
- **建议使用**：首次使用前运行一次
- **下载一次**：模型会缓存，不需要重复下载

**工作流**：
```
[PSD 模型预下载器]
  下载档位: 标准档
  包含补全模型: 否
  ↓
  下载日志
  状态预览
```

---

## 🚀 快速开始

### 新手推荐流程

**第一步：预下载模型（首次使用）**
1. 添加"PSD 模型预下载器"节点
2. 选择"标准档 (2.4GB)"
3. 执行节点，等待下载完成（约 5-10 分钟）

**第二步：使用智能分层**
1. 添加"智能分层 PSD 生成器"节点
2. 连接生图节点的输出
3. 设置参数：
   - 分层档位：标准档
   - 检测文字：是
   - 最小元素面积：1000
4. 执行节点，生成分层 PSD

**第三步：在 Photoshop 中编辑**
1. 打开生成的 PSD 文件
2. 查看图层面板
3. 编辑各个图层

## 💡 常见问题

### Q1: 首次使用很慢？
**A**: 首次使用会自动下载模型（2-5GB），建议先运行"PSD 模型预下载器"。

### Q2: 三个档位如何选择？
**A**: 
- 简单商品图 → 经济档
- 复杂场景、海报 → 标准档 ⭐
- 商业级、被遮挡补全 → 极致档

### Q3: 需要同时使用三个节点吗？
**A**: 不需要。
- 只需要 PSD 格式 → 只用"PSD 文件保存器"
- 需要分层 → 只用"智能分层 PSD 生成器"
- 预下载器是可选的

### Q4: 模型下载到哪里？
**A**: 
- SAM2: `~/.cache/sam2/`
- EasyOCR: `~/.EasyOCR/`
- rembg: `~/.u2net/`
- LaMa: `~/.cache/simple_lama_inpainting/`

### Q5: 如何查看下载进度？
**A**: 查看 ComfyUI 控制台输出，会显示下载进度。

## 🔧 技术细节

### 分层原理

**经济档**：
1. rembg 去除背景
2. OpenCV 连通域检测元素
3. 生成图层

**标准档**：
1. SAM2 精准分割
2. EasyOCR 识别文字
3. 智能分组图层

**极致档**：
1. SAM2 精准分割
2. LaMa 补全被遮挡区域
3. 生成完整背景层

---

**提示**：所有节点都支持自动安装依赖，首次使用会自动下载所需的库和模型。
