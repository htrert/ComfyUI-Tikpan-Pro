**📋 图生视频操作步骤（以 HappyHorse I2V 为例）**

**Step 1：搭建工作流**

`配图5：ComfyUI 工作流截图：LoadImage → 首帧图片 → HappyHorse I2V → PreviewVideo`

```
[LoadImage: 你的商品图] → [首帧图片]
[HappyHorse 1.0 I2V 图生视频] → [PreviewVideo]
```

**Step 2：填参数**

- **api_key**：填攀升AI密钥
- **prompt**：描述视频里要发生什么（关键！）
- **mode**：同步（等结果）or 异步（先提交再查）
- **resolution**：分辨率，测试用低档
- **duration**：视频时长，5秒/8秒/10秒

`配图6：节点参数填写截图，圈出 prompt 输入框`

**Step 3：写视频提示词**

图生视频的提示词，重点描述**动作和运镜**，不是产品本身

万能模板：
```
镜头[运镜方式]，产品周围[环境变化]，
[光效描述]，整体保持产品结构不变，
真实摄影风格，商业大片质感。
```

举例：
- `镜头缓慢推进，香水瓶周围轻微水汽流动，柔和暖光，不改变瓶子外观`
- `镜头绕产品环绕旋转360度，场景有细小光粒飘落，背景渐变模糊`
- `镜头由远及近慢速推进，产品静止，背景有微风吹过绿植效果`

`配图7：提示词填写示例截图`

**Step 4：运行等待**

点 Queue Prompt，等结果
同步模式一般等1-3分钟

`配图8：视频生成完成后的 PreviewVideo 节点播放截图`

**⚡ 异步模式怎么用（更稳定）**

视频生成时间长，建议用**异步模式**：

1. mode 选"异步"
2. 提交后会返回一个**任务ID**
3. 把任务ID接到「异步任务查询与下载」节点
4. 等几分钟再查，不用一直盯着

`配图9：异步工作流截图：HappyHorse → 任务ID → 异步任务查询与下载 → PreviewVideo`

这样你可以同时提交多个任务
一次批量出5条视频

### 图生视频详解（I2V 节点）

**工作流搭建：**

```
[LoadImage: 商品图] → [首帧图片]
[HappyHorse 1.0 I2V] → [PreviewVideo]
                      → [保存视频节点]
```

**核心参数：**

| 参数 | 推荐值 | 说明 |
| --- | --- | --- |
| prompt | 详细描述动作 | 这是最关键的参数，决定视频效果 |
| mode | 测试用同步，批量用异步 | 视频生成慢，建议异步 |
| resolution | 720p → 1080p | 先低档测试 |
| duration | 5s / 8s / 10s | 时间越长费用越高 |
| max_wait_seconds | 300-600 | 同步等待的最长时间 |

`配图：节点参数完整截图，带中文标注`

**视频提示词技巧：**

图生视频的提示词和图片提示词逻辑不同，核心是描述"动态变化"：

1. 描述运镜方式（推进/环绕/下摇/上移）
2. 描述环境变化（光影变化/粒子飘落/液体流动）
3. 锁定不变的元素（产品本身要写"保持不变"）

**高分提示词结构：**
```
[镜头运镜] + [环境氛围变化] + [产品锁定声明] + [整体风格]
```

实战例子：
> 镜头缓慢向前推进，停留在产品特写，香水瓶周围有轻微水汽雾气流动，背景灯光柔和地随时间渐变，始终保持产品外观、颜色、logo完全不变，真实商业摄影风格，高质量。

### 同步 vs 异步，怎么选？

**同步模式**
- 提交后节点等待直到视频生成完成
- 适合：单个测试任务
- 风险：等待时间超过 max_wait_seconds 会报超时

**异步模式（推荐正式使用）**
- 提交后立即返回任务ID
- 用「异步任务查询与下载」节点查询结果
- 适合：批量任务、长视频

异步工作流：
```
[HappyHorse I2V: mode=异步] → task_id
[Tikpan 异步任务查询与下载] → [PreviewVideo]
```

```
参考图2是人物，保持人物特征和妆容一致；
参考图3是背景场景。
生成一段5秒钟的产品展示视频，人物拿着产品展示，镜头缓慢推进。
```

### 视频质量提升技巧

1. **首帧图质量要高**：分辨率越高，生成的视频越清晰
2. **提示词不要太复杂**：动作描述简洁，不要超过3个动作指令
3. **先测5秒**：确认效果后再出10秒，省费用
4. **异步批量**：同时提交3-5个不同提示词，选最好的一条
