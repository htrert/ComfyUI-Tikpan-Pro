### 节点选择

本期用到的节点：

**主力：GPT-Image-2 官方修图 V2（`TikpanGptImage2OfficialEditV2`）**

这是 OpenAI 官方 GPT-Image-2 模型的修图版本，通过攀升AI中转接入，走的是 `/v1/images/edits` 接口。

主要能力：
- 背景替换（保留产品精准性高）
- 风格迁移（在保留产品的前提下改风格）
- 遮罩局部重绘（精准指定修改区域）
- 多参考图引导（最多传入4张风格/场景参考图）

**辅助：Gemini 14图极限生图（`TikpanGeminiImageMaxNode`）**

当你需要把多张参考图融合在一起（比如：一张人物参考 + 一张服装参考 + 一张背景参考）时用这个，最多可传14张参考图。

### 工作流搭建

**换背景工作流（最简单版本）：**

`配图：完整工作流截图，节点连线清晰可见`

```
[LoadImage: 商品原图] → [主图像]
[GPT-Image-2 官方修图 V2] → [PreviewImage]
                           → [SaveImage]
```

**带遮罩的精准修图工作流：**

```
[LoadImage: 商品原图] → [主图像]
[MaskEditor: 画遮罩] → [遮罩掩码]
[GPT-Image-2 官方修图 V2] → [PreviewImage]
```

遮罩规则：白色 = 允许修改；黑色 = 锁定保留
如果结果刚好反了，打开节点里的「遮罩反相」开关。

`配图：遮罩示例截图，展示白色区域（背景部分）和黑色区域（产品部分）的区别`

### 参数详解

| 参数 | 建议设置 | 说明 |
| --- | --- | --- |
| 编辑指令 | 详细描述 | 越具体效果越好 |
| 分辨率档位 | 测试用1K，出片用2K/4K | 越高越慢越贵 |
| 画面比例 | 电商主图用1:1，竖版海报用9:16 | 按用途选 |
| 背景模式 | opaque（不透明背景） | 换实体背景时用这个 |
| 提示增强 | 建议开启 | 提高稳定性 |
| 跳过错误 | 批量任务时开启 | 失败不中断整个流程 |

### 多参考图进阶：Gemini 14图

当你有具体的场景参考时，用 Gemini 14图极限生图会比纯文字描述稳定得多。

操作方式：
1. 准备3-5张参考图（可以是你喜欢的广告大片、场景截图等）
2. 分别接入参考图1、参考图2……
3. 提示词里说明每张图的用途

提示词示例：
> 参考图1是产品，请保持产品外观完全一致；参考图2是你希望的背景氛围；参考图3是灯光风格。生成一张商业产品海报，竖版9:16，高端感，真实摄影。

### 常见问题

**Q：换背景后产品变形了怎么办？**
A：提示词里加一句"保留产品外观、比例、所有细节完全不变"，同时可以试试画遮罩把产品锁定。

**Q：生成的图分辨率不够高？**
A：分辨率档位调到2K或4K，注意费用也会对应增加。

**Q：总是被审核拦截？**
A：审核强度选择"auto"或"low"，提示词避免出现类似烟草、违规品牌等词。
