**📋 Step by Step**

**Step 1：准备原材料**

`配图3：LoadImage 节点加载商品图的截图`

一张清晰的商品图即可
（白底最好，方便 AI 分离产品和背景）

**Step 2：并行启动图片精修**

`配图4：GPT-Image-2 修图V2 节点参数截图`

一次生成4张，每张用不同的编辑指令：
- 指令1：高端大理石台面+暖光
- 指令2：户外自然场景
- 指令3：极简渐变纯色背景
- 指令4：你品牌色系的搭配方案

技巧：用 ComfyUI 的**批量运行**功能同时运行

**Step 3：图生视频**

`配图5：HappyHorse I2V 节点工作流截图`

把精修后最好看的一张图接入视频节点
写2条不同的视频提示词（分别出2条视频）

提示词1（产品展示）：
> 镜头缓慢推进至特写，产品周围有轻微光粒飘散，暖光柔和，整体保持产品外观不变，商业质感

提示词2（场景沉浸）：
> 镜头从远景缓慢拉近，背景有微风流动效果，产品稳定在画面中央，自然光照射，高端感

mode 选**异步**，提交完先去做别的

**Step 4：生成背景音乐**

`配图6：Suno 节点参数截图`

和视频节点同时提交，节省等待时间

提示词：
> 适合美妆/护肤品牌展示视频的背景音乐，风格优雅时尚，有少许节奏感，整体轻盈明亮，长度约30秒

**Step 5：生成口播旁白**

`配图7：豆包TTS 节点参数截图`

提前写好旁白文案，粘入节点

旁白模板：
> 这款[产品名]，专为[目标人群]打造<#0.3#>
> 每一次使用，都是[核心卖点]<#0.5#>
> 现在，属于你

**Step 6：查询异步任务结果**

`配图8：异步任务查询与下载节点截图`

等大约5-10分钟后
把视频和音乐的任务ID接入查询节点
下载所有生成好的文件

### 六期内容的终极目标

经过前五期的铺垫，今天这期把所有模块整合成一条完整的工作流。

**目标场景：** 你是一名电商卖家/内容创作者，手里有一个产品，需要准备：
- 多风格商品精修图（用于主图/详情页/社媒投图）
- 商品展示视频（用于短视频平台投放）
- 背景音乐（用于视频配乐）
- 旁白配音（用于视频口播）

**传统方案：** 摄影+后期+拍摄+剪辑+配乐+配音 → 最少3天，成本2000元+

**AI方案：** 一套 ComfyUI 工作流 → 15分钟，成本个位数

`配图：整体成品素材包截图展示`

### 完整工作流架构

`配图：完整工作流全图，清晰展示4条支线的节点连接关系`

核心思路：把单个节点的工作**并行化**，不是一步一步等，而是同时提交多个任务，最后统一收货。

**工作流四条支线：**

1. **图片支线**：商品图 → 多风格精修（换背景、换场景、换风格）
2. **视频支线**：精修图 → 展示视频（异步提交，边等边做其他）
3. **音乐支线**：Suno 音乐生成（同时提交，不占等待时间）
4. **配音支线**：TTS 旁白生成（几秒钟完成，最快）

---

### 工作流搭建步骤

**Phase 1：图片处理（约2-5分钟）**

```
[LoadImage: 商品原图]
         ↓

  实例1：高端台面场景
  实例2：户外自然场景
  实例3：极简纯色背景
  实例4：品牌色系场景
         ↓
[PreviewImage / SaveImage: 保存所有结果]
```

`配图：4个修图节点并排的工作流截图`

**Phase 2：视频生成（异步，约5-15分钟）**

```
[LoadImage: 最佳精修图]
         ↓
[HappyHorse I2V 节点 × 2]
  节点1：镜头推进版
  节点2：环绕旋转版
  mode: 异步
         ↓
[task_id × 2 保存起来]
```

异步提交后立即去执行 Phase 3 和 4，充分利用等待时间。

**Phase 3：音乐生成（异步，约3-8分钟）**

```

  创作提示词: 品牌风格音乐描述
  模型: chirp-v5
  mode: 异步
```

**Phase 4：配音生成（同步，约10-30秒）**

```

  合成文本: 提前写好的旁白文案
  音色: 选好适合品牌调性的音色
  音频格式: mp3
```

**Phase 5：收货**

```
[Tikpan 异步任务查询与下载] × 2
  分别查视频 task_id
  下载视频文件

[PreviewAudio / SaveAudio]
  预听、保存音乐和配音
```

### 参数快速配置表

**图片节点（GPT-Image-2 修图V2）：**

| 参数 | 建议值 |
| --- | --- |
| 分辨率档位 | 2K（精修出图） |
| 画面比例 | 1:1 主图 / 9:16 竖版海报 |
| 背景模式 | opaque |
| 生成张数 | 1-2张/实例 |

**视频节点（HappyHorse I2V）：**

| 参数 | 建议值 |
| --- | --- |
| mode | 异步 |
| resolution | 1080p |
| duration | 8s 或 10s |
| max_wait_seconds | 1200 |

**Suno 音乐节点：**

| 参数 | 建议值 |
| --- | --- |
| 生成模式 | 普通生成 |
| 模型版本 | V5 最新通用 |
| 生成纯音乐 | 开启（背景音乐不需要人声） |

**豆包TTS 节点：**

| 参数 | 建议值 |
| --- | --- |
| 语速 | 0.9（稍慢，有品质感） |
| 音频格式 | mp3 |
| 音色 | 根据品牌调性选 |

### 进阶思路：批量内容矩阵

掌握了基础工作流后，下一步是规模化：

**SKU批量处理：**
每个产品准备一张原图，工作流里用批量参数，一次处理5-10个SKU。

**多平台适配：**
用同一条视频素材，图片节点切换比例（1:1主图/9:16小红书/16:9横版），一次出图覆盖多个平台。

**A/B测试：**
用并发生图引擎，同一产品用不同提示词同时出多张，快速测试哪种风格效果更好。
