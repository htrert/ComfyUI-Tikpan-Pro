### 前言：为什么要用这个插件？

如果你刚开始接触 ComfyUI，或者你已经在用 AI 工具但觉得"太分散、太麻烦"，这篇文章值得你花10分钟读完。

`配图：同小红书版，展示生图+生视频+音频的成品效果对比图`

攀升AI（Tikpan）是一个 AI API 聚合平台，汇集了国内外顶尖的生成式 AI 模型。而今天要介绍的 **ComfyUI-Tikpan-Pro 插件**，就是把这些模型能力直接接入 ComfyUI 工作流的官方节点包。

简单说：装一个插件，你就同时拥有了：
- OpenAI GPT-Image-2 的图像生成能力
- Google Veo 3.1 / Gemini Omni 的视频生成能力
- Suno 的 AI 作曲能力
- MiniMax / 豆包 的语音合成能力
- Gemini 3.5 Flash 的多模态分析能力
- ……以及更多

---

### 节点功能全览

插件安装后，在 ComfyUI 的节点菜单里会出现「👑 Tikpan 官方独家节点」分类，下面按功能分为6个子类：

`配图：ComfyUI 节点分类截图，展示完整的6个子分类`

**01 图片 Image**
包含 GPT-Image-2 官方生图/修图、Gemini 14图极限生图、Grok Imagine Image 系列、Qwen-Image-2.0、Wan 2.7 Image Pro、豆包图像生成等14个图片类节点。

**02 视频 Video**
包含 HappyHorse 文生/图生/参考生/编辑视频、Grok-Videos、Veo 3.1 多模型、Kling Motion Control 动作控制、Vidu3 系列、Gemini Omni Flash 等11个视频类节点。

**03 音频 Audio**
包含 Suno 音乐生成、MiniMax speech-2.8-hd/turbo、豆包语音合成 2.0、Gemini 3.1 Flash TTS 共5个音频类节点。

**04-05 多模态与分析**
GPT-5.4 Mini、Gemini 3 Flash Preview、Gemini 3.5 Flash、Grok多图剧本重构，专门用来理解图片/视频/文档内容。

**06 任务工具**
异步任务查询、API多模型并发引擎、异步图片任务组，用于商业批量工作流。

### 安装步骤（详细版）

**方式一：通过 ComfyUI Manager 安装（推荐）**

1. 打开 ComfyUI，在侧边栏找到 Manager
2. 点击「自定义节点管理器」
3. 搜索框输入 `Tikpan`
4. 找到 ComfyUI-Tikpan-Pro，点击 Install
5. 安装完成后重启 ComfyUI

`配图：ComfyUI Manager 操作步骤截图（带箭头标注）`

**方式二：手动安装**

1. 下载插件压缩包
2. 解压到 `ComfyUI/custom_nodes/` 目录下
3. 重启 ComfyUI 即可

---

### 获取 API Key

`配图：攀升AI官网控制台界面截图（打码敏感信息）`

1. 访问攀升AI官网，注册账号
2. 进入控制台 → 创建 API Key
3. 复制密钥，填写到 ComfyUI 节点的 `API_密钥` 输入框中

**计费说明：**
- 充值方式友好，支持人民币直充，约0.6元兑1美元余额
- 各模型按调用次数或 token 数计费
- 测试时建议先用低分辨率、短视频档位，确认效果后再升级参数
