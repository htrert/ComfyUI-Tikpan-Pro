**🎙️ 二、TTS 口播配音（文字→语音）**

做短视频经常需要旁白
之前找配音演员，现在直接用 AI 合成

我用得最多的是**豆包语音合成 2.0**
中文效果相当自然，100多个音色可选

`配图4：豆包 TTS 节点截图，展示音色下拉选项`

`配图5：合成出来的音频在 PreviewAudio 播放的截图`

操作步骤：

1. 右键 → 音频｜豆包语音合成 2.0
2. 合成文本：把你的旁白文字粘进去
3. 音色：从下拉里选，每个音色有中文名+风格描述
4. 语速/音调：根据内容调整
5. 音频格式：mp3（兼容性最好）

**几个好用的音色：**
- 旁白/解说类：找"播音"或"主持人"风格
- 少女活泼：找"甜美女声"或"活泼女声"
- 品质感：找"成熟男声"或"低沉磁性"

### 音频是短视频被忽视的核心竞争力

很多人做短视频，把大量时间花在画面上，却用一首随便找的背景音乐，或者磕磕碰碰的真人录音。

好的音频往往是观看完播率的关键因素。今天这期，教你用攀升AI节点解决两个核心音频需求：
1. 用 Suno 生成带人声的背景音乐
2. 用 TTS 生成自然的口播旁白

### 一、Suno 音乐生成

**节点：Suno 音乐生成（`TikpanSunoMusicNode`）**

**核心参数：**

| 参数 | 说明 | 推荐 |
| --- | --- | --- |
| 生成模式 | 普通生成/续写/歌手风格 | 新歌选"普通生成" |
| 创作提示词 | 描述你要什么风格的歌 | 越具体越好 |
| 风格标签 | 写流派关键词 | pop, cinematic, lofi 等 |
| 模型版本 | V5/Fenix/V4 | V5 效果最新 |
| 生成纯音乐 | 开关 | 不要人声时打开 |

节点一次生成**两首候选歌曲**，各有差异，选最合适的。

**高质量提示词公式：**
```
[情感基调] + [使用场景] + [语言要求] + [节奏特点] + [特殊要求]
```

实例：
> 一首充满希望感的中文流行电子歌，适合美妆品牌宣传视频开头，副歌朗朗上口，整体明亮积极，BPM 中等偏快，结尾有渐强感。

**风格标签参考：**
- 流行系列：`pop`, `mandopop`, `city pop`
- 电子系列：`edm`, `electronic`, `synth`
- 影视感：`cinematic`, `orchestral`, `epic`
- 国风：`chinese traditional`, `guzheng`
- 轻松系：`lofi`, `acoustic`, `folk`

`配图：Suno 节点完整参数截图，带中文标注`

**续写功能：**

如果你之前有一首 Suno 生成的歌曲，想续写或者调整，
- 填入 `续写_歌曲ID`（从任务ID输出里获取）
- 填入 `续写起始秒数`
- 继续生成，在原歌的基础上延伸

### 二、豆包语音合成 2.0

**节点：豆包语音合成 2.0（`TikpanDoubaoTTS20Node`）**

豆包语音合成是字节跳动的 TTS 模型，中文发音自然，音色选择丰富，非常适合：
- 短视频旁白/解说
- 产品介绍口播
- 电商直播脚本预听
- 有声内容制作

**核心参数：**

| 参数 | 说明 |
| --- | --- |
| 合成文本 | 粘贴你的旁白文字，短文本 < 1万字符 |
| 音色 | 下拉选，每个音色附中文名+风格说明 |
| 语速 | 0.5-2.0倍，默认1.0 |
| 音量 | 0.5-2.0倍，默认1.0 |
| 音调 | -12到12半音，0是原调 |
| 情感 | 保持默认不传即可 |
| 音频格式 | mp3（前台播放用），wav（后期处理用） |

**停顿控制小技巧：**

在文本里加 `<#秒数#>` 插入停顿：
- `<#0.3#>` = 0.3秒停顿
- `<#0.8#>` = 0.8秒停顿

示例：
```
欢迎来到今天的分享<#0.3#>
今天我要为大家介绍这款改变我日常护肤的产品<#0.5#>
它就是……
```

### 三、MiniMax speech-2.8-hd（高端配音）

当对配音质量要求更高时，用 MiniMax speech-2.8-hd。

音色细节更丰富，支持：
- 发音字典（自定义特殊词发音）
- 音色混合（两种音色按比例混合）
- 字幕输出（配合视频字幕使用）
- 异步接口（长文本用异步提交更稳）

`配图：MiniMax speech 节点截图，展示高级参数区域`

### 四、Gemini 3.1 Flash TTS（多语言）

出海内容首选，官方30个预置音色，每个都有英文名称+中文风格说明。

额外功能：
- 语气指令：可以用自然语言描述语气，如"自信、商业旁白风格"
- 语言代码选择：zh-CN、en-US、ja-JP 等
- 输入/输出 token 用量可用于网站计费统计

`配图：Gemini TTS 节点截图，圈出语气指令输入框`

### 音乐+配音组合使用

生成完两个音频后，在视频剪辑软件中：
1. 将口播配音音轨设为主音轨（音量80-100%）
2. 将背景音乐音轨设为副音轨（音量20-30%）
3. 适当淡入淡出，让两者自然融合

`配图：剪映双音轨编辑界面示意（配音在上，音乐在下）`

### 商业化提示

如果你是用来给客户做内容的：
- `计费字符数` 输出可以直接用于记录成本
- 豆包 TTS 按字符计费，MiniMax 也有对应计费字段
- 建议把音色名称和 voice_type 分开记录，方便后续维护
