### 1. 分辨率payload终极修复

**根因确认**（通过截图对比+官方API文档）:
- 别人的Zhenzhen节点用 `/v1/images/generations` 端点，`resolution` + `size` 是标准参数，2K/4K完美生效
- 我们的节点用 `/v1/chat/completions` 端点，分辨率没有标准位置可放，中转站不认

**官方API文档确认**（apimart.ai）:
- `/v1/images/generations` 是图像生成正规端点
- 参数：`model`, `prompt`, `size`(比例), `resolution`(1K/2K/4K), `n`, `image_urls`
- 返回：`{ "data": [ { "b64_json": "..." } ] }`
- flash 和 pro 模型都支持2K/4K
